## 🎉 Let's go!

Para iniciar o projeto insira o comando <code>npm run dev</code>

## 💻 Projeto

Uma aplicação para simular um banco com entradas e saídas, além de buscas e novas transações.


## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- HTML e CSS
- Typescript
- React
- Node e NPM
- Vite


## 📚 Bibliotecas

- Styled Components
- Radix UI
- Axios
